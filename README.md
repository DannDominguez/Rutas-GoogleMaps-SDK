# Rutas-GoogleMaps-SDK

Proyecto en Swift y SwiftUI para grabado de rutas y persistencia de las mismas.

## Tecnologías

- Google Maps SDK
- XCode 15.3
- Swift 5.0 
- Core Data


## Agregar la vista de nuestro mapa a SwiftUI
```javascript
struct ContentView: View {
var body: some View {
        MapViewControllerBridge()
    }
}
```
